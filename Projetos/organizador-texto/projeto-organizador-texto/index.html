Bem-vindo ao Organizador de Texto!

Para usar o sistema, siga estas instruções:

  1. Digite ou cole seu texto na área de texto.
  2. Clique no botão "Organizar" para formatar o texto.
  3. Se o sistema detectar erros, siga as orientações.
  4. Clique no botão "Limpar" para limpar a área de texto.

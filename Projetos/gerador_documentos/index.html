Projeto Amigo Secreto

Aplicação para sorteio de amigo secreto, onde os participantes podem adicionar seus nomes e realizar o sorteio garantindo que ninguém sorteie a si mesmo.

Ver Projeto

O Projeto Amigo Secreto foi desenvolvido como parte do desafio Alura. Ele permite que os usuários adicionem seus nomes e realizem o sorteio de forma justa, garantindo que ninguém sorteie a si mesmo. A aplicação foi construída utilizando HTML, CSS e JavaScript, com foco em usabilidade e experiência do usuário.

Boas Práticas em Desenvolvimento

Seguir boas práticas é essencial para manter um código limpo e eficiente. Aqui estão algumas recomendações:

  • Escrever código bem estruturado e documentado.
  • Seguir princípios de desenvolvimento como SOLID.
  • Manter um fluxo de versionamento eficiente com Git.
  • Utilizar testes automatizados sempre que possível.

Metodologias Ágeis e Scrum

Scrum é um framework ágil que ajuda na organização e entrega eficiente de projetos. Principais conceitos:

  • Sprints: Períodos curtos de desenvolvimento com metas definidas.
  • Daily Stand-ups: Reuniões rápidas para alinhar tarefas.
  • Backlog: Lista organizada de tarefas pendentes.
